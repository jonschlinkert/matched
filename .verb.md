## Usage

```js
var glob = require('{%= name %}');
```

**async**

```js
glob(['*.js'], function(err, files) {
  //=> ['utils.js', 'index.js']
});
```

**sync**

```js
var files = glob.sync(['*.js']);
//=> ['utils.js', 'index.js']
```

**options**

Both sync and async take options as the second argument:

```js
glob(['*.js'], {cwd: 'test'}, function(err, files) {
  //=> ['test.js']
});
```
