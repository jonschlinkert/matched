## Usage

```js
var glob = require('{%= name %}');
```

**async**

```js
glob(['*.js'], function(err, files) {
  //=> ['utils.js', 'index.js']
});
```

**sync**

```js
var files = glob.sync(['*.js']);
//=> ['utils.js', 'index.js']
```

**promise**

```js
glob.promise(['*.txt'])
  .then(function(files) {
    //=> ['a.txt', 'b.txt', 'c.txt']
  });
```

**options**

All methods take options as the second argument:

```js
glob(['*.js'], {cwd: 'test'}, function(err, files) {
  //=> ['test.js']
});
```
